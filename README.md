### Clone repository
```bash
# Clone this repository
$ git clone https://github.com/joaolobao380/Proffy.git
```


### Install Front
```bash

# Go into the web path
$ cd Proffy/web

# Install dependencies
$ yarn install

# Run the Front
$ yarn start
```

### Install Back
```bash

# Go into the server path
$ cd Proffy/server

# Install dependencies
$ yarn install

# Run the Front
$ yarn start
```
### Run Migrate
```bash

# Run migrate
$ yarn knex:migrate
```
